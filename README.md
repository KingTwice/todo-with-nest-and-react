# Simple To-do list app  

## Stack  

### Frontend  
-React  
-Typescript  

### Backend  
- Typescript  
- NestJs  
- MongoDB  
- Mongoose      
--------

Both frontend and backend need a .env file  
example:   
-Backend:   
```  
MONGO_USERNAME=<your_username> 
MONGO_PASSWORD=<your_password>  
```
-Frontend:   
```  
REACT_APP_API_URL=<your_backend_api_url>
```
